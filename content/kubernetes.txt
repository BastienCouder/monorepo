Déployer sur Kubernetes
Assurez-vous d'avoir un cluster Kubernetes configuré (ex: via Ansible ou un service cloud comme GKE, EKS, AKS).

Appliquez les configurations Kubernetes :

kubectl apply -f k8s/web-deployment.yaml
kubectl apply -f k8s/web-service.yaml
kubectl apply -f k8s/software-deployment.yaml
kubectl apply -f k8s/software-service.yaml

Vérifiez les déploiements :

kubectl get pods
kubectl get services

Accéder aux applications
Next.js sera disponible sur l'IP externe du service Kubernetes (ex: http://<cluster-ip>:80).
React Vite sera disponible sur l'IP externe du service Kubernetes (ex: http://<cluster-ip>:80).
